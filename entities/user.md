Пользователь - это минимальная единица организации. Создаётся при регистрации пользователя. При регистрации указывается:

ФИО ( Иванов Иван Иванович )
Фракция ( Человек | Киборг | Соулер )
Род деятельности ( Разработчик | Дизайнер | Архитектор)
Легенда ( Пишется либо самим пользователем, либо генерируется ИИ )
